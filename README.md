# Messenger App

Minimalistic messenger web app for real-time communication.

Features:
- real time messaging
- group support

### Backend:
- Express
- Socket.io
- PostgreSQL
- knex
- Jest

### Frontend:
- React
- Redux
- TailwindCSS
- Vitest + testing-library

### How to run

```
cd client
pnpm i
pnpm build

cd ../server
pnpm i
pnpm start
```
